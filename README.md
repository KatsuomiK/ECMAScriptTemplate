# ECMAScript 2015 Project Template

Watchify + Babelify + BrowserSync  

This base project (excluding lib directory) is public domain.

## Setup

$ npm install

This project uses following tools.

- browserify
- watchify
- babelify
- browser-sync
- parallelshell
- uglifyjs
- mocha

## Development

To launch a browser and start watch:

$ npm run watch

To just launch:

$ npm start

To build and minify:

$ npm run build
